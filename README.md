# Power-factor-correction-analysis
25 Days of MATLAB Power Systems Journey

This repository contains my 25day learning journey exploring core concepts of Electrical Power Systems using MATLAB.
The goal of this project was to move from basic concepts to advanced visualization and simulation, while building consistency and practical understanding.

What This Project Covers
Over 25 days, I worked on:

 Fundamentals
- Power (Real, Reactive, Apparent)
- Power Factor and its importance
- Losses and efficiency
  
 Analysis Concepts
- Harmonics and FFT
- Total Harmonic Distortion (THD)
- Signal analysis
  
 Three-Phase Systems
- Balanced systems
- Unbalanced systems
- Line vs Phase voltages
  
 Advanced Concepts
- Phasor representation
- Rotating magnetic field
- Space vector concept
  
 Visualization
- Waveform plotting
- Frequency spectrum graphs
- Phasor animations
- System comparison graphs


 Project Structure
 Day 01 to Day 25 mfiles followed by the images and videos of graphs and simulation.
